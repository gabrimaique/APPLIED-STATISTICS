# Applied Statistics

This repository contains submissions for the Applied Statistics module at Atlantic Technological University (ATU) 2025.

## Quickstart

- The main file is a Jupyter notebook

## About

All submissions for module assessments are included here.

## Contents

- `problems.ipynb` - Jupyter notebook with problem solutions.
- `.gitignore` - Tells Git what files to ignore.
- `requirements.txt` - List of required Python packages.

## Getting Started

### Clone the Repository

https://github.com/gabrimaique/APPLIED-STATISTICS.git


### Install Dependencies

- Python 3.7 or higher (download from [python.org](https://www.python.org/downloads/))
- Install required packages:

pip install -r requirements.txt


### Run the Notebook

- Start Jupyter:

jupyter notebook problems.ipynb
